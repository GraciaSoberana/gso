## 1. Logo

### Variantes oficiales

| Variante | Uso recomendado |
|---|---|
| **Color (gradiente)** | Fondos blancos o claros, portadas, materiales premium |
| **Plano blanco** | Fondos oscuros, proyección, camisetas oscuras, sellos |
| **Plano negro** | Documentos en escala de grises, impresión económica |

### Elementos del logo

El logo de GSO tiene **dos componentes inseparables**:

1. **Símbolo** — Forma circular con motivo interior (sello/escudo estilizado)
2. **Logotipo** — Texto "GRACIA SOBERANA ORIZABA" en caps

Nunca usar el símbolo solo ni el texto solo como representación oficial de la iglesia.

### Espacio de protección

Mantener un margen mínimo equivalente a la altura de la letra "G" del logotipo alrededor de todo el logo. No colocar texto, imágenes ni elementos decorativos dentro de esa zona.

### Usos prohibidos

- ❌ Deformar o estirar el logo
- ❌ Cambiar los colores por otros no definidos en esta guía
- ❌ Agregar efectos (sombra, bisel, resplandor, contorno)
- ❌ Colocar sobre fondos que generen bajo contraste
- ❌ Rotar el logo

## 2. Paleta de Colores

### Colores primarios

Extraídos del gradiente oficial del logo.

#### Verde Teal — Color de identidad principal

```
HEX     #015A6C
RGB     1, 90, 108
CMYK    99, 17, 0, 58
HSL     191°, 98%, 21%
```

Evoca profundidad, firmeza y fidelidad. Es el color estructural de la identidad: fondos principales, encabezados, barras de sección.

#### Terracota Rosada — Color de acento

```
HEX     #CD9888
RGB     205, 152, 136
CMYK    0, 26, 34, 20
HSL     14°, 40%, 67%
```

Evoca calidez, cercanía y gracia. Usada para acentos, líneas decorativas, resaltados y detalles secundarios.

---

### Colores secundarios

Derivados de los primarios para uso en documentos y materiales extendidos.

#### Verde Teal Claro

```
HEX     #E6F2F5
RGB     230, 242, 245
```

Para fondos de secciones, callouts, fondos de tablas alternas.

#### Terracota Suave

```
HEX     #F5EAE7
RGB     245, 234, 231
```

Para fondos de citas, alertas suaves, destacados de devoción.

#### Blanco

```
HEX     #FFFFFF
```

Fondo estándar de documentos, diapositivas, áreas de texto.

#### Gris Texto

```
HEX     #2D2D2D
RGB     45, 45, 45
```

Texto corrido principal. No usar negro puro (#000000) para texto en documentos.

#### Gris Suave

```
HEX     #6B7280
RGB     107, 114, 128
```

Textos secundarios, notas al pie, etiquetas de sección, metadatos.

## 3. Tipografía

### Fuente principal — Inter

Inter es la tipografía oficial de GSO. Es una sans-serif diseñada para legibilidad óptima en pantalla y papel.

**Pesos en uso:**

| Peso | Nombre | Uso |
|---|---|---|
| 300 | Light | Citas largas, texto de apoyo |
| 400 | Regular | Texto corrido |
| 500 | Medium | Subtítulos de sección |
| 600 | SemiBold | Títulos secundarios, énfasis |
| 700 | Bold | Títulos principales, encabezados |

**Descarga:** [rsms.me/inter](https://rsms.me/inter) · Google Fonts · o variable font `Inter.ttf`

---

### Jerarquía tipográfica

#### En documentos (Word / PDF)

| Nivel | Tamaño | Peso | Color |
|---|---|---|---|
| Título de documento | 28–32 pt | Bold | `#015A6C` |
| Encabezado H1 | 20–22 pt | Bold | `#015A6C` |
| Encabezado H2 | 16–18 pt | SemiBold | `#015A6C` |
| Encabezado H3 | 13–14 pt | SemiBold | `#2D2D2D` |
| Texto corrido | 11–12 pt | Regular | `#2D2D2D` |
| Cita bíblica | 11–12 pt | Light / Italic | `#2D2D2D` |
| Nota / secundario | 9–10 pt | Regular | `#6B7280` |

#### En diapositivas (Proyección / Advanced Slides)

| Elemento | Tamaño aprox. | Peso |
|---|---|---|
| Título de portada | 52–60 pt | Bold |
| Versículo principal | 36–42 pt | Light o Regular |
| Título de sección | 32–38 pt | Bold |
| Texto de cuerpo | 24–28 pt | Regular |
| Notas / referencia | 18–20 pt | Light |

---

### Fuente alternativa

Si Inter no está disponible, usar **Calibri** (Word) o **Lato** (web/Google Slides) como sustituto aceptable.

Nunca usar: Comic Sans, Times New Roman, Papyrus, ni fuentes decorativas no autorizadas.

## 4. Tono y Voz

La comunicación de GSO refleja su identidad teológica y pastoral.

### Principios de voz

- **Clara** — Sin jerga interna ni tecnicismos innecesarios. El evangelio es para todos.
- **Cálida** — Habla como una comunidad, no como una institución.
- **Seria sin ser fría** — La gracia soberana es doctrina que transforma; comunicarla con peso y con gozo.
- **En español neutro** — Accesible a cualquier hispanohablante, sin regionalismos que excluyan.

### Tono por contexto

| Contexto | Tono |
|---|---|
| Sermones y materiales de enseñanza | Expositivo, pastoral, estructurado |
| Redes sociales | Cálido, breve, invitacional |
| Boletín / anuncios | Informativo y fraternal |
| Correspondencia oficial | Formal pero cercano |
| Materiales para niños | Simple, narrativo, lleno de color |

## 5. Iconografía y Elementos Gráficos

### Estilo de íconos

Usar íconos de línea (outline), **no** íconos rellenos sólidos. Grosor de trazo: 1.5–2 px. Estilo recomendado: Heroicons, Feather Icons, o Lucide.

### Líneas decorativas

La línea horizontal delgada en color `#CD9888` (terracota) funciona como separador de sección elegante y coherente con la identidad.

### Formas geométricas

Usar rectángulos y formas simples. Evitar elementos muy elaborados, gradientes complejos externos al logo, o texturas que distraigan del contenido.

### Fotografía

- Preferir fotografías propias de la comunidad
- Tono cálido (no filtros fríos o desaturados)
- Evitar imágenes de banco que parezcan genéricas o corporativas
- Para portadas: composición con espacio negativo donde insertar texto

## 6. Aplicaciones Específicas

### Diapositivas de proyección

- Fondo oscuro: `#015A6C` con texto blanco
- Fondo claro: `#FFFFFF` con texto `#015A6C`
- Logo en esquina superior derecha o inferior derecha
- Versículos: Inter Light, color blanco o terracota suave
- Transiciones: cortas, sin efectos llamativos

### Documentos pastorales (PDGs, guías, cartas)

- Encabezado con logo color alineado a la izquierda
- Línea divisoria en `#CD9888` debajo del encabezado
- Fuente Inter 11–12 pt para texto
- Numeración de página en gris suave

### Redes sociales

- Formato cuadrado 1:1 para Instagram / WhatsApp
- Formato 16:9 para historias o banners
- Mantener logo visible, nunca cortado
- Usar la paleta de colores oficial; no improvisar combinaciones

### Materiales impresos (volantes, programas)

- Impresión a color: usar paleta oficial completa
- Impresión en blanco y negro: logo plano negro, eliminar elementos de color
- Tamaño mínimo de logo en impresión: 2.5 cm de ancho

## 7. Qué No Hacer

| ❌ Prohibido | ✅ Correcto |
|---|---|
| Usar colores no definidos en esta guía | Usar siempre los HEX oficiales |
| Colocar texto sobre el logo | Respetar el espacio de protección |
| Usar más de 2–3 fuentes en un mismo documento | Usar Inter en sus variantes de peso |
| Saturar con demasiados elementos visuales | Diseño limpio, con aire y jerarquía |
| Usar gradientes no derivados de la paleta | Solo el gradiente oficial del logo |
| Traducir el nombre de la iglesia en materiales gráficos | Siempre "Gracia Soberana Orizaba" |
